Q7. Count	consonants	in	a	string	(recursive	methods)
Given	a	string,	count	total	number	of	consonants	in	it.	A	consonant	is	a	
English	alphabet	character	that	is	not	vowel	(a,	e,	i,	o	and	u).	Examples	
of	constants	are	b,	c,	d,	f,	g,	..


Examples	:
Input	:	abc de	Output	:	3	
There	are	three	consonants	b,	c	and	d.	
Input	:	geeksforgeeks portal	Output	:	12
